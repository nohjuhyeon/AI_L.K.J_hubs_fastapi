## - 팀원별 느낀점
- 노주현

<p>PM으로서 전체적인 일정을 계획하는 데 어려움을 많이 느낀 프로젝트였다. 돌이켜 생각해보면 중, 장기 계획을 세우고 진행했더라면 더 수월한 프로젝트가 됐을 것 같다. 이번 경험을 토대로, 앞으로는 프로젝트를 시작하기에 앞서, 중, 장기적 목표를 세우고 계획을 세워야겠다.</p>

- 김명준
<p>3차 프로젝트에서는 페르소나를 선정하고 이에 대한 마케팅 전략을 수립하는 과정을 거치며, 이전에는 생각하지 못했던 다양한 관점을 고민하고 탐구할 수 있었습니다. 이러한 과정은 저를 데이터 분석가로서 한 단계 더 성장시키는 중요한 계기가 되었습니다. 이러한 모든 경험들이 모여 이번 최종 발표에서 더욱 의미 있는 결과를 도출할 수 있었고, 앞으로의 데이터 분석 여정에 큰 밑거름이 될 것이라 확신합니다.</p>

- 문석천
<p>페르소나를 선정하고 이에 대한 마케팅 방안 전략을 준비하는 과정을 통해서 한걸음 더 분석 방향성에 대해 고민할 수 있는 경험이었다고 생각한다. 또한 분석을 한 결과를 업그레이드 시키기 위해서 추가적인 데이터 분석이 끊임없이 이루어진다는 것을 배움으로써 분석가로서의 창의적인 비지니스 마인드와 판단력이 중요한 것을 배울 수 있었다.</p>

- 장영지
<p>이번 프로젝트를 통해 분석 방향성을 설정하는 것이 데이터 분석에서 얼마나 중요한지 알게 되었다. 페르소나를 잘 설정하면 분석 방향성이 명확해지고, 더 유의미한 인사이트를 도출할 수 있음을 깨달았다. 또한, 데이터 분석 자체보다 그에 맞는 데이터를 수집하는게 더 어렵다는 생각이 들었다. 이런저런 분석 아이디어는 많았으나 데이터 확보에 어려움이 있어 분석하지 못한 것이 아쉽다.</p>
